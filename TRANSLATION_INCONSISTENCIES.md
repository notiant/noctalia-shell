# Translation Period Inconsistencies - Detailed Report

This report lists all translation strings that end with a period (`.`) but don't end with a period in `en.json`.
For `zh-CN.json`, it looks for the Chinese period (`。`) instead of `.`

---

- `de.json`: 14 issues
